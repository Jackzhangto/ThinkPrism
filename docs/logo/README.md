# ThinkPrism Logo 资源

本目录存放 ThinkPrism 品牌 Logo 及设计依据，供扩展图标、商店素材、文档使用。

## 文件说明

| 文件 | 说明 |
|------|------|
| `design-philosophy.md` | Logo 设计哲学（Prism Mind），依据 canvas-design 技能生成，与《原型设计》§8.1 配色与方向一致 |
| `ThinkPrism-logo.png` | Logo 主图：棱镜折射光线汇聚成思维气泡，深蓝 #1E3A8A + 荧光青 #22D3EE，可用于图标与商店 |

## 使用建议

- **扩展图标**：可从本 PNG 裁切为 48×48、128×128 等规格，或基于同一设计重导出多尺寸。
- **商店与文档**：直接引用 `ThinkPrism-logo.png` 或按需缩放。

配色与理念详见《原型设计》文档 **八、品牌与上线待办 → 8.1 Logo 与配色建议**。
