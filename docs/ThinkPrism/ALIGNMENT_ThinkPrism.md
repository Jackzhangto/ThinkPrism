# ALIGNMENT_ThinkPrism

## 1. 项目目标

开发 **ThinkPrism**，一款基于 Google Chrome Manifest V3 的浏览器扩展。
**核心价值**：基于《AI 思维模式激活指令手册 (Lyra Edition · V3.0)》，让用户在与 ChatGPT、Claude、DeepSeek 等 AI 对话时，能够通过热键式无缝切换，快速激活多层级思维模式指令（L1 微指令、L2 黄金剧本、L3 百科模式、持久化设定），提升对话质量与深度。
**技术特性**：纯本地运行（无后端）、隐私安全、React + Vite + TypeScript 技术栈。

## 2. 核心需求共识 (基于 Claude SPAC 文档融合)

| 模块 | 需求描述 | 关键验收标准 |
| :--- | :--- | :--- |
| **思维模式管理** | 支持 L1/L2/L3/持久化 四层级模式浏览与检索。 | - Tab 分类展示<br>- 实时搜索过滤 (标题/Emoji/Tag)<br>- 占位符 `[内容]` 自动替换为选中文本 |
| **指令注入** | 自动将生成的指令注入到当前 AI 网页的输入框。 | - 支持白名单站点自动注入<br>- 失败自动重试 (3次)<br>- 不支持站点/失败时回退到剪贴板并 Toast 提示 |
| **场景推荐** | 基于用户当前场景 ("产品"/"代码"/"决策"/"学习") 推荐模式。 | - Popup 顶部常驻场景按钮<br>- 点击展示 1-3 个推荐模式<br>- 数据驱动推荐 (基于 Tag) |
| **后续建议** | 指令执行后，再次打开提供后续行动建议。 | - 记录上次执行时间<br>- 间隔 > 3分钟显示建议卡片 (如 "元认知自检") |
| **持久化模式** | 设定全局的人设/风格，自动追加到所有指令前。 | - 存储于 sync storage<br>- 每次注入自动拼接到最前<br>- 全局唯一激活 |
| **组合指令** | 支持多选模式，组合生成长指令。 | - 组装顺序：持久化 -> L3 -> L2 -> L1<br>- 双换行符分隔<br>- 提供清空按钮 |
| **新手引导** | 首次安装后的引导流程。 | - 首次打开显示欢迎卡片<br>- 推荐 3 个入门模式<br>- 完成一次注入后不再显示 |
| **选项配置** | 管理白名单站点、自定义模式、语言设置。 | - 编辑/新增白名单 (URL/Selector)<br>- 导入/导出/编辑自定义模式 (JSON)<br>- 中英文切换 |
| **非功能需求** | 性能、隐私、安全。 | - Popup 冷启动 < 200ms<br>- 零数据收集 (无后端)<br>- 最小权限原则 (storage, activeTab, scripting) |

## 3. 技术栈决策 (已确认)

- **核心框架**: React 18+
- **构建工具**: Vite 5+ (配合 @crxjs/vite-plugin)
- **语言**: TypeScript 5+
- **样式**: Tailwind CSS
- **状态管理**: Zustand (轻量级，适合管理模式数据和用户偏好)
- **测试**: Vitest + fast-check (用于核心逻辑的属性测试)
- **规范**: Manifest V3

## 4. 关键术语表

- **L1 微指令**: 单条思维标签，即插即用。
- **L2 黄金剧本**: 多步骤思维链触发指令。
- **L3 百科模式**: 完整长指令框架。
- **持久化模式**: 全局生效的人设设定。
- **Selector**: 用于定位网页输入框的 CSS 选择器。
