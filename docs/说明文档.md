# 项目说明文档

## 1. 项目规划
### 1.1 项目简介
ThinkPrism 是一款开源 Chrome 浏览器插件，基于《AI 思维模式激活指令手册 (Lyra Edition · V3.0)》，旨在帮助用户在与 AI 对话时快速激活高阶思维模式。

### 1.2 核心目标
- 实现多层级指令（微指令、黄金剧本、百科模式、持久化设定）的快速调用。
- 智能适配主流 AI 聊天页面（ChatGPT, Claude, DeepSeek 等）。
- 提供模式推荐引擎与执行反馈机制。

### 1.3 时间规划
- **Phase 1 (Align & Architect)**: 需求对齐与架构设计 [已完成]
- **Phase 2 (Atomize & Approve)**: 任务拆分与计划确认 [已完成]
- **Phase 3 (Automate)**: 核心功能开发 (Popup, Content Script, Storage, Options) [已完成]
- **Phase 4 (Assess)**: 测试验收与发布准备 [已完成]

## 2. 实施方案
采用 **6A 工作流** (Align, Architect, Atomize, Approve, Automate, Assess) 进行开发。
- **技术栈**: React 18 + Vite 5 + TypeScript + Tailwind CSS (Manifest V3)
- **核心库**: Zustand (状态管理), Lucide React (图标), clsx (类名合并)
- **测试**: Vitest + fast-check (属性测试), @testing-library/react (组件测试)

## 4. 更新日志
### 2026-03-03 ThinkPrism v1.0.3 Feature Update (Auto-Inject Toggle)
- **功能增强**: 恢复自动注入功能，并将其与用户设置中的 **“自动注入内容脚本”** 开关绑定。
  - **开关开启 (ON)**: 点击模式卡片，提示词将**直接注入**到当前网页的输入框中（如 ChatGPT, Claude, 千问等）。
  - **开关关闭 (OFF)**: 点击模式卡片，提示词将**仅复制到剪贴板**，并提示“已复制指令”，方便用户手动粘贴校验。
- **文档说明**: 解答关于《AI 思维模式激活指令手册》提示词在大模型间的通用性问题。

### 2026-03-03 ThinkPrism v1.0.2 Hotfix (Manual Mode)
- **功能变更**: 响应用户需求，暂时禁用自动注入功能，改为 **“点击即复制”** 模式。
  - 目的：方便用户手动将提示词粘贴到大模型输入框中，以校验提示词内容的正确性。
  - 行为：点击任何模式（如红队），提示词将自动复制到剪贴板，并弹出“已复制指令”提示。

### 2026-03-03 ThinkPrism v1.0.1 Hotfix
- **Bug 修复**: 修复在千问等大模型网站输入框中，插入指令后导致无法使用退回键 (Backspace) 和中文输入法失效的问题。
  - 原因：原有的 `document.execCommand` 插入方式与现代前端框架 (React/Vue) 的状态管理冲突，且插入后未正确恢复焦点。
  - 解决：重构 `InjectionService.ts`，优先使用原生 Setter 触发 React 状态更新，并强制恢复焦点。
- **文档更新**: 更新 FINAL_ThinkPrism.md 记录修复内容。
